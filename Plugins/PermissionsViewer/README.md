# PermissionsViewer - [Download](https://betterdiscord.app/Download?id=29)

Allows you to view all the permissions for a user, guild, or channel!

https://github.com/user-attachments/assets/f0a192dd-90c4-4162-9600-2bc2e84824db

## Features

### Popouts

See a list of all the server permissions a user has in their popout.

![Popouts](https://github.com/user-attachments/assets/78c329e0-40ed-4c8e-b59c-6b296ae8159c)

### Member Permissions

Clicking the details button in the popout or using the context menu will bring up a modal which has a role-by-role breakdown of which roles give what permissions.

![User](https://github.com/user-attachments/assets/a9cd5ef8-35fa-446e-8839-1b9cb1dc8962)

### Server Explorer

Using the context menu on a guild can give you a full role permission explorer.

![Guild](https://github.com/user-attachments/assets/5f8f3ada-7447-496d-a2e1-2592c2fc25d9)

### Channel Overwrites

View all the specific overwrites for the channel.

![Channel](https://github.com/user-attachments/assets/232c194b-910a-41ba-ad15-3433fb34fb07)
